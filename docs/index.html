Hacker News の仕組み
Points（得点）とは？
ユーザーが記事を upvote（賛成票）した合計数です。多くの人が「面白い・有益だ」と評価した記事ほど高い得点になります。
トップページのランキング基準
得点が多く新しい記事ほど上位に表示され、時間が経つと自然に順位が下がります。
このサイトのデータ取得タイミング
毎朝 JST 8:00 に Hacker News API からその時点のトップ30記事を取得しています。リアルタイムではなく、朝8時時点のスナップショットです。
