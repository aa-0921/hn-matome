## 重要な前提確認

GitHubは2025年12月に価格改定を発表。2026年1月からpublicリポジトリにも
$0.002/分のコントロールプレーン費用が発生する可能性があります。
ただし「public リポジトリは完全無料（月2000分）」という情報もあり矛盾。
**GitHub の Settings > Billing で現在の実際のコストを確認することを推奨します。**

もし有料化されていても月4時間実行なら約$5程度なので、テンプレート販売などの収益モデルは十分成立します。

### 2位: 業界ニュース日本語AI自動集約ニュースレター

> Hacker News・Reddit・海外テックニュースをGitHub Actions cronで毎日自動収集し、
> Claude APIで日本語要約→無料メール配信。スポンサー枠で収益化。

| 項目 | 内容 |
|------|------|
| ターゲット | 日本人エンジニア・個人開発者（英語ニュースを追いたいが時間がない） |
| 収益モデル | 無料配信 → スポンサー枠（企業向け）¥50,000〜¥150,000/月 |
| 想定月額 | 無料（読者向け）/ 有料はスポンサー企業向け |
| MVP要件 | ①Hacker News自動取得 ②Claude要約 ③メール配信（Resend使用） |
| 横展開の型 | 読者が増えたら→有料Proプラン（全記事・カスタマイズ）→¥980/月 |

**5軸評価**

| 評価軸 | スコア | 理由 |
|--------|:------:|------|
| タイミング   | 5/5 | AI要約ニュースレターが世界的トレンド・日本語版ほぼ存在しない |
| 実現可能性   | 4/5 | HN MCP・Claude APIで4週間以内に構築可能 |
| 収益性       | 4/5 | 読者1000人超えればスポンサー収益が発生・月¥50,000〜 |
| 競合優位性   | 4/5 | 既存の日本語テックニュースは手動運営・AI自動化は差別化 |
| 個人開発適性 | 3/5 | 週1回の品質チェックは必要・ただし大部分は自動 |
| **合計**     | **4.2/5.0** | |

**実際の不満・ニーズの声**
- "毎日のHacker Newsを追うのに1時間かかる" — 日本エンジニア
- "Beehiiv等は英語UIで使いにくい" — 個人開発者ブログ調査より
- GAS×GeminiでAIニュースレターを自作した記事が人気（GAS代替としてのニーズ）

**競合**: Beehiiv・Substack（英語のみ）、企業向けブラストメール（月¥5,000〜高額）
弱点: 日本語対応なし・個人向け無料プランなし

**技術スタック（全部無料）**

```yaml
# .github/workflows/newsletter.yml
on:
  schedule:
    - cron: '0 7 * * 1'  # 毎週月曜朝7時
jobs:
  send:
    runs-on: ubuntu-latest
    steps:
      - HN MCP でトップ20取得
      - Claude API で日本語要約生成
      - Resend API でメール配信
      - GitHub commit でアーカイブ保存
```
